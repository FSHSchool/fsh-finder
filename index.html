Is your IG missing?

  • The easiest way to add it is to register it for CI builds on build.fhir.org.
  • You can also submit a pull request to add your IG's GitHub repository to the list of manually crawled sources. You can also have every repository in your GitHub org/username manually crawled if you have multiple IGs using FSH. Currently the manually crawled sources are defined here.
  • If your IG is registered for CI builds or is already covered by the manual crawl sources, and it still does not appear, please log an issue.
